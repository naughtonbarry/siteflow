Where the data comes from
The app is built around swappable adapters. To add a paid traffic source later (e.g. DataForSEO for real monthly visits), implement fetchData(domain) returning the standard schema and register it — no rebuild of the UI needed.
Adapter interface
// Each source implements this shape:
const MyAdapter = {
  name: 'DataForSEO',
  provides: ['Monthly visits', 'Traffic sources'],
  async fetchData(domain) {
    const r = await callApi(domain);        // server-side / keyed
    return { monthlyVisitors: r.visits };    // -> standard schema
  }
};
// Registered adapters are merged per site in loadSite().
