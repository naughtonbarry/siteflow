To add a new data source, implement the DataSourceAdapter interface and register it in the sources registry. See architecture notes below.
Architecture — adding a new data source
// 1. Implement the DataSourceAdapter interface
class MyNewSource {
  name = 'MyNewSource';
  provides = ['monthly_traffic', 'domain_authority'];

  async fetchData(url) {
    // Call your API here
    const raw = await fetch(`https://api.mynewsource.com/site?q=${url}`);
    // Normalize to standard schema
    return {
      monthlyVisitors: raw.monthly_visits,
      domainAuthority: raw.da_score,
    };
  }

  normalize(raw) { /* map to standard schema */ }
}

// 2. Register in DATA_SOURCES array (top of dashboard() function)
dataSources.push(new MyNewSource());

// 3. Merge in aggregateCompetitorData()
// The aggregator automatically merges all enabled sources.

Connect Google Analytics
This dashboard uses the Google Analytics Data API (GA4). You'll need a service account or OAuth credentials.
Note: In production, credentials should be stored server-side. This demo stores them in memory only and never sends them to third parties.
